Currency Input Mask

A practical guide to implementing formatted currency inputs using jQuery Inputmask & Bootstrap 5. Covers USD, EUR, IRR, TRY, GBP, JPY, AED and more.

Get Started Live Demos

Quick Start

Three steps to get currency input masks running in your project.

1
Include Dependencies

Add these CDN links inside your HTML. jQuery must load before Inputmask.

<!-- CSS -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">

<!-- JS (place before </body>) -->
<script src="https://cdn.jsdelivr.net/npm/jquery@3.7.1/dist/jquery.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/inputmask@5.0.9/dist/inputmask.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/inputmask@5.0.9/dist/jquery.inputmask.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
2
Add an Input Element

Create a standard text input with an id or class you can target.

<input type="text" id="price" class="form-control" placeholder="Enter amount">
3
Initialize the Mask

Call Inputmask with the currency alias and your preferred options.

$(document).ready(function () {
  $("#price").inputmask("currency", {
    prefix: "$ ",
    groupSeparator: ",",
    radixPoint: ".",
    digits: 2,
    digitsOptional: false,
    rightAlign: false,
    autoGroup: true,
    removeMaskOnSubmit: true
  });
});
Tip: Use removeMaskOnSubmit: true so the server receives a clean numeric value instead of the formatted string. Alternatively, use autoUnmask: true to always keep the underlying value clean.

Configuration Reference

Key options for the Inputmask currency / numeric alias.

Option Type Default Description
alias string "numeric" Base alias to use. Set to "currency" for currency formatting.
prefix string "" String prepended to the value (e.g., "$ ", "€ ").
suffix string "" String appended to the value (e.g., " USD", " ﷼").
groupSeparator string "" Thousands separator ("," for US/UK, "." for EU).
radixPoint string "." Decimal point character ("." for US/UK, "," for EU).
digits number 2 Number of fractional digits. Set to 0 for currencies without decimals (e.g., JPY, IRR).
digitsOptional boolean true When false, trailing zeros are always shown (e.g., 5.00).
autoGroup boolean false Automatically insert groupSeparator as the user types.
rightAlign boolean true Align input text to the right (common in accounting UIs).
autoUnmask boolean false When true, .val() returns the raw unmasked value.
removeMaskOnSubmit boolean false Strip mask formatting when the form is submitted.
numericInput boolean false Numeric mode — digits are entered from right to left.
allowMinus boolean true Allow negative values.
min number — Minimum allowed value.
max number — Maximum allowed value.
placeholder string "0" Placeholder character for empty positions.
For the full list of options, see the official Inputmask documentation.
